Aplicación Android nativa que implementa el clásico juego del ahorcado, con una interfaz simple e interactiva.

El objetivo es adivinar la palabra oculta seleccionando letras, gestionando intentos y visualizando el progreso del juego en tiempo real.

## 🚀 Características

* 🎮 Lógica completa del juego del ahorcado
* 🔤 Selección de letras interactiva
* ❤️ Sistema de intentos / vidas
* 📊 Actualización dinámica del estado del juego
* 🔁 Reinicio de partida
* 📱 Interfaz simple y clara

---

## 🧠 Tecnologías

* **Kotlin**
* **Android SDK (XML / Views)**
* **Lógica de estado en tiempo real**
* **Material Design**

## 🏗️ Estructura

El proyecto está organizado en:

* `Activity` principal → manejo del juego
* `UI XML` → interfaz y layout
* `Game Logic` → reglas, validaciones e intentos

Esto permite mantener separada la lógica del juego de la interfaz.
